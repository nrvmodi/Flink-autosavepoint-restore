# Flink-autosavepoint-restore
This project contains script file which will take save point for your flink job and restore it with new Flink job

There are multiple placeholder and variable that you need to replace

flinkBucketPath: This is the path to save flink's savepoint.
flinkWebUiUrl: This is the web url of Flink dashboard.
<<Replace_job_name>>: This is the placeholder to get job id for the given jobname.
SAVEPOINT: This is the placeholder which is placed on quota-deployment.yaml file.
